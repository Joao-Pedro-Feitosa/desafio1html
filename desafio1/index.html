Listas

Início

As listas servem para organizar informações de forma mais clara. Existem dois tipos: listas ordenadas, que usam números, e listas não ordenadas, que usam bolinhas ou marcadores. Cada item da lista fica separado, o que ajuda muito quando queremos mostrar passos ou tópicos.

Títulos

Início

Os títulos vão do h1 até o h6. O h1 é o mais importante e geralmente aparece no topo da página, já o h6 é o menor. Eles servem para organizar os assuntos, como se fossem os capítulos e subtítulos de um livro, ajudando o leitor a entender o conteúdo.

Links

Início

Os links servem para levar o usuário de um lugar para outro. Eles podem apontar para outra página, para outro site ou até para uma parte diferente da mesma página. Também é possível fazer links para abrir arquivos, enviar e-mails ou até para abrir em uma nova aba do navegador.

Textos

Início

As tags de texto servem para escrever o conteúdo principal da página. O parágrafo é o mais usado, mas também existem outras opções, como deixar palavras em negrito, em itálico ou até criar citações. Isso ajuda a dar destaque e organizar melhor a leitura.

Inputs

Início

Os inputs são usados em formulários para que o usuário possa digitar ou escolher informações. Eles podem ser de vários tipos, como caixa de texto, senha, botão de seleção, caixa de marcação e até para enviar arquivos. São muito importantes quando queremos coletar dados do usuário.

Imagens

Início

As imagens deixam a página mais visual e interessante. Para usá-las, precisamos indicar o endereço do arquivo da foto. Também podemos colocar uma descrição que aparece caso a imagem não carregue e que ajuda pessoas com deficiência visual. As imagens podem ter diferentes tamanhos e também podem ser ajustadas dentro do site.
